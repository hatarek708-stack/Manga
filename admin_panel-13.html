بروكسي CORS لتجاوز حظر Cloudflare

المواقع العربية محمية بـ Cloudflare. لتجاوز هذا الحظر تماماً، يوصى بإنشاء Cloudflare Worker مجاني واستخدامه كبروكسي خاص بك [1, 2].

طريقة إنشاء البروكسي الخاص بك في دقيقة واحدة مجاناً:
  1. قم بإنشاء حساب مجاني في dash.cloudflare.com
  2. اذهب إلى Compute (Workers & Pages) واضغط على Create Application ثم Create Worker.
  3. قم بتسميته واضغط Deploy ثم اضغط Edit Code.
  4. امسح الكود القديم تماماً وضع هذا الكود بدلاً منه:
export default {
  async fetch(request, env, ctx) {
    // Handle CORS preflight
    if (request.method === 'OPTIONS') {
      return new Response(null, {
        headers: {
          'Access-Control-Allow-Origin': '*',
          'Access-Control-Allow-Methods': 'GET, POST, OPTIONS',
          'Access-Control-Allow-Headers': '*',
        }
      });
    }

    const reqUrl = new URL(request.url);
    const url = reqUrl.searchParams.get('url');
    if (!url) return new Response('Missing ?url= parameter', { status: 400 });

    // إزالة _cb و _site من URL الهدف (كانا للمنع من الكاش فقط)
    let targetUrl = url;
    try {
      const u = new URL(url);
      u.searchParams.delete('_cb');
      u.searchParams.delete('_site');
      u.searchParams.delete('_t');
      targetUrl = u.toString();
    } catch(e) {}

    try {
      // تمرير method و body كما هي (يدعم POST للبحث عبر admin-ajax)
      const fwdHeaders = {
          'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/125.0.0.0 Safari/537.36',
          'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,*/*;q=0.8',
          'Accept-Language': 'ar,en-US;q=0.9,en;q=0.8',
          'Accept-Encoding': 'gzip, deflate, br',
          'Referer': new URL(targetUrl).origin + '/',
          'Cache-Control': 'no-cache, no-store',
          'Pragma': 'no-cache',
      };
      const fetchInit = {
        method: request.method,
        headers: fwdHeaders,
        redirect: 'follow',
        cf: { cacheTtl: -1, cacheEverything: false, bypassCache: true },
      };
      // مرّر جسم الطلب وترويسة نوع المحتوى عند POST (مطلوب للبحث wp-manga-search-manga)
      if (request.method === 'POST' || request.method === 'PUT') {
        fetchInit.body = await request.text();
        fwdHeaders['Content-Type'] = request.headers.get('Content-Type') || 'application/x-www-form-urlencoded; charset=UTF-8';
        // طلبات admin-ajax تتطلب أحياناً ترويسة X-Requested-With
        fwdHeaders['X-Requested-With'] = 'XMLHttpRequest';
      }
      const res = await fetch(targetUrl, fetchInit);

      const newHeaders = new Headers(res.headers);
      newHeaders.set('Access-Control-Allow-Origin', '*');
      newHeaders.set('Access-Control-Allow-Methods', 'GET, POST, OPTIONS');
      newHeaders.set('Cache-Control', 'no-store');
      newHeaders.delete('Content-Security-Policy');
      newHeaders.delete('X-Frame-Options');
      newHeaders.delete('Set-Cookie');

      return new Response(res.body, { status: res.status, headers: newHeaders });
    } catch (err) {
      return new Response('Worker fetch error: ' + err.message, { status: 502 });
    }
  }
}
  1. اضغط Save and Deploy في الأعلى.
  2. انسخ رابط الـ Worker الخاص بك وضعه في الحقل أدناه متبوعاً بـ ?url= (مثال: https://my-worker.workers.dev/?url=) واضغط حفظ.
